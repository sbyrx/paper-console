# PC-1 Frontend

This directory contains the PC-1 settings UI built with React, Vite, and Tailwind CSS.

Use a Unix-like shell for project commands. On Windows, run them from WSL.

## Development

```bash
cd web
npm install
npm run dev
```

The dev server runs at [http://localhost:5173](http://localhost:5173).

## Build

```bash
cd web
npm run build
```

The production build output is written to `web/dist`.

See the repo root `readme.md` for full project setup, backend commands, and testing workflows.
